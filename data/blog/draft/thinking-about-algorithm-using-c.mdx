3. https://www.acmicpc.net/problem/9093
   -> c++을 사용하면 한줄 입력을 받을때 getline을 사용할 수 밖에 없다
   -> 이때 처음에 받는 n과 같은 test case를 받는 경우는 한 줄이 아닌 띄어쓰기 단위로 받는다(>> 연산자를 통해서)
   -> 이 다음에 getline 함수를 사용하게 되면 \n 이 버퍼에 남아서 getline함수가 \n을 받게 되므로 예상하지 못한 입력이 받아진다
   -> 이를 해결하기 위해서 cin.ignore()라는 함수를 사용하면 버퍼를 clear해준다.
